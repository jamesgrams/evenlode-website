Work

Websites

  • Game 103
    Game 103 is a site that hosts family-friendly games and entertainment. It consists of a database of games, including many created by Evenlode, with features such as a star rating system, social media sharing, advanced searching, game recommendations based on play history, and and more.
  • Making Do Recipes
    A recipe finder, featured by the FPIES Foundation, for people with severe allergies that allows people to share their own recipes. This website lists the allergens in each of its ingredients, and allows people to enter either their safe foods or their allegens. From there, the website will tell them what recipes are safe for them. Recipes can list substitutes for ingredients, and people can enter an allergen tolerance count if they want to try their own substitutes.
  • Racecourse
    A simple, easy-to-learn online course content manager that is used currently by both churches and schools. It allows instructors to post content to classes they set up, and students to view that content.

Tools

  • Instagram Poster
    A somewhat popular automation tool that posts an image to instragram programitically. It can be easily set up to post an image at a specific point in time.
  • Budgeteer
    Budgeteer is a budgeting program with a fun twist. It connects to your bank account, and lists purchases as coins. You can create treasure chests, which represent budget categories, to put your coins in to.
  • Ringleader
    Ringleader is a tool that allows users to run web-browser automation programs using the blockchain-based distributed computing system of iExec RLC.

Home Media

  • GuyStation
    GuyStation is home media software that primarly serves as an emulator frontend allowing for classic video games to be played easily with any controller or keyboard. However, it can be accessed by any device with a web browser for not just control, but remote streaming. This allows you to play games on any device connected to your network. It has also a built-in media player (that can stream content remotely) and web browser.
  • Papa Voice
    Papa Voice is a custom tool I made for my grandfather to control his TV with with his voice using commands such as "Watch Golf" or "Watch Fox Business". It works by looking up the TV guide online, and then matching phrases it receives from the microphone to those found in the TV guide listings. It communites with an IR blaster to change the channel appropriately.

Apps

  • IPAide
    IPAide is a tool for linguistic students that assists them with understanding and referncing the phoenetic alphabet.
  • Animals of the Bible
    Animals of the Bible is a tool that lists all the animals of the Bible and provides references to what verses they are found in.

Browser Extensions

  • Pull Up Flash
    Pull Up Flash lets people easily access Flash content on the web by clicking it to open it in Flash Player. Since the Flash Plugin for web has been discontinued, this provides a one-click way to continue to play Flash games and movies.
  • Gamepad to Keyboard Mapper
    This extension allows users to map buttons on their gamepads to keys on their keyboard. This allows them to use controllers for browser games and applications that are only set up to work with keyboard controls.
  • Food Cub
    Food Cub is a chrome browser extension built for people with allergies that highlights items that contain certain ingredients that the user chooses (e.g. corn) on Food Lion's and Aldi's websites. This way, people can know what to avoid without having to click on each item and read the ingredients.
